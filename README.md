# Test task  "The bookseller has many books"

Книготорговець має безліч книг, класифікованих за 26 категоріями з позначками A, B, ... Z. Кожна книга має код c із 3, 4, 5 або більше символів. Перший символ коду - це велика літера, яка визначає категорію книги.

У списку книжок продавця книг кожен код супроводжується пробілом і цілим числом n (int n> = 0), що вказує на кількість книг цього коду на складі.

Наприклад, витяг зі списку запасів може бути:
L = ["ABART 20", "CDXEF 50", "BKWRK 25", "BTSQZ 89", "DRTYM 60"] 

Вам буде наданий список запасів (наприклад: L) і список категорій великими літерами, наприклад:
M = ["A", "B", "C", "W"]
і ваше завдання - знайти всі книги L з кодами, що належать до кожної категорії M, і підсумувати їх кількість відповідно до кожної категорії.

Для списків L і M, наприклад, вам потрібно повернути рядок:
(A : 20) - (B : 114) - (C : 50) - (W : 0)
де A, B, C, W - категорії, 20 - сума унікальної книги категорії A, 114 - сума, що відповідає "BKWRK" і "BTSQZ", 50, що відповідає "CDXEF", і 0 - категорії "W" оскільки немає коду, що починається на W.
